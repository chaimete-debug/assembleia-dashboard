# Dashboard Assembleia Distrital

## Estrutura pronta para Vercel

Este pacote já está com a estrutura correcta:

- index.html
- manifest.json
- offline.html
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

## Como publicar

1. Extraia este ZIP.
2. Envie todos os ficheiros para a raiz do repositório no GitHub.
3. No Vercel, ligue o projecto a esse repositório.
4. Framework Preset: Other
5. Root Directory: ./
6. Build Command: deixar vazio
7. Output Directory: deixar vazio
